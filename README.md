## Description

A backend API for a meal management application that allows users to create, read, update, and delete
meal addons.

## Installation

```bash
$ npm install
```
## Migration

```bash
# import modules of database schema
$ npm run Migration

```


## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
